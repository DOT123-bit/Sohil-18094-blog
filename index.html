Study of Hash Functions

Understanding RSHash and its application to different data types

What is RSHash?

RSHash is a simple non-cryptographic hash function created by Robert Sedgwicks. It's designed for use in hash tables and provides a good distribution of hash values.

The algorithm processes each character in the input string, using bitwise operations to combine them into a final hash value.

While not suitable for cryptographic purposes, RSHash is efficient for general hashing needs and demonstrates the fundamental principles of hash functions.

In this vlog, we'll explore how RSHash processes different types of data commonly found in internet technology.

RSHash Algorithm

The RSHash algorithm can be expressed with the following formula:

hash = (hash * a + char) % b

Where:

  • a and b are carefully chosen constants
  • char is the current character being processed
  • hash is initialized to 0

The algorithm processes each character in sequence, updating the hash value with each iteration.
